# 🧺 Laundry Mart Website

Laundry Mart is a simple and user-friendly laundry service website developed using HTML.  
This project is designed to provide information about laundry services, pricing details, and allow customers to book services online.

---

## 📌 Project Overview

The website includes:

- Welcome section with introduction
- Service list display
- Price list with Normal and VIP pricing
- Booking form (Name, Email, Phone)
- Contact information
- Attractive laundry-themed image

This project helps beginners understand the structure of a basic service-based website.

---

## 🛠️ Technologies Used

- HTML5
- Basic Web Design Structure

---

## 💡 Features

✔ Clean and simple layout  
✔ Easy to understand code structure  
✔ Beginner-friendly project  
✔ Can be hosted using GitHub Pages  

---

## 🎯 Purpose of Project

This project was created for learning and practice purposes.  
It demonstrates how to design a static business website using basic HTML concepts like:

- Headings
- Paragraphs
- Lists
- Tables
- Forms
- Images
